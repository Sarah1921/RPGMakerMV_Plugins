# Plugins

## [Utility](utility)
 - イベント条件として使うことを想定したユーティリティ関数群。
 - 「特定の時間帯限定」「特定の曜日限定」といったイベントが作成しやすくなる。

## [Cache control](cachecontrol)
 - ゲームのWeb公開時に、キャッシュ残りによって変更がされない現象を防止するプラグイン。
