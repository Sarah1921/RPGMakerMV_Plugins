# Plugins

## [ユーティリティ関数群 Utility](utility)
 - イベント条件として使うことを想定したユーティリティ関数群。
 - 「特定の時間帯限定」「特定の曜日限定」といったイベントが作成しやすくなる。

## [キャッシュ制御 Cache control](cachecontrol)
 - Web公開しているゲームを更新した時に「キャッシュ残りによって更新が反映されない」といったことを防止するプラグイン。
